## 🚀 Next Steps (In Order)

### Step 1: Install & Run (2 minutes)

```bash
cd /Users/user/Documents/axoblade/axoblade.com
npm install
npm run dev
```

Open http://localhost:3000 in your browser → See your portfolio live!

### Step 2: Customize Content (15 minutes)

Read **QUICKSTART.md** and update:

- [ ] Your name and title

- [ ] About section

- [ ] Projects

### Step 3: Deploy to Production (5 minutes)

Choose one:

- **Vercel** (recommended): `npm i -g vercel && vercel`
